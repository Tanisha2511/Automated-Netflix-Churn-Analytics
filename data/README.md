# CSV Data Description

This folder contains the CSV files used in the Netflix Churn Analysis project.

| CSV File Name               | Source / SQL View            | Description |
|-----------------------------|------------------------------|-------------|
| v_netflix_clean.csv         | v_netflix_clean             | Cleaned base dataset with all raw features standardized. |
| v_netflix_features.csv      | v_netflix_features          | Feature-engineered table with engagement and satisfaction levels. |
| v_netflix_segments.csv      | v_netflix_segments          | Segmented customers: high, medium, low churn risk. |
| v_kpi_overview.csv          | v_kpi_overview              | KPI summary: total users, churn rate, average engagement & satisfaction. |
| v_retention_targets.csv     | v_retention_targets         | Top 50 high-risk customers for retention targeting. |
